# PWA Icons

This directory contains the Progressive Web App icons for Digital Footprints.

## Required Sizes

Generate icons in the following sizes:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Design Guidelines

- **Theme**: Eco-friendly, green technology
- **Icon Style**: Leaf emoji üçÉ or stylized leaf design
- **Background**: White or transparent
- **Primary Color**: Green (#22c55e)

## Generation

You can use tools like:
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Figma/Sketch/Adobe XD with export presets

Or run this command with a source icon:
```bash
npx pwa-asset-generator source-icon.png public/icons --icon-only --background "#ffffff"
```
